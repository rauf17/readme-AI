# Design System: Vercel/GitHub Dark Mode

## 1. Visual Theme
A modern developer-focused dark UI inspired by GitHub, Vercel, and VS Code. The design prioritizes high readability, low eye strain, sharp contrast, subtle depth, and a minimal but premium feel. The interface uses a deep dark base with soft elevation layers and subtle accent colors for interaction feedback.

## 2. Color Palette
**Primary Background**
- Base: `#0d0f14` (deep dark, not pure black)
- Surface 1: `#161b22` (Cards and panels)
- Surface 2: `#1c2330` (Secondary elevated surfaces)

**Text Colors**
- Primary Text: `#e6edf3`
- Secondary Text: `#9da7b3`
- Muted Text: `#6b7280`

**Accent Colors (Interactive)**
- Primary Accent: `#3b82f6` (blue like Vercel/GitHub)
- Success: `#22c55e`
- Warning: `#f59e0b`
- Error: `#ef4444`

**Borders**
- Default: `#2a3441`
- Hover: `#3a4658`

## 3. Typography
**Font Family**
- UI: `Inter`, `system-ui`, `sans-serif`
- Code/Editor: `JetBrains Mono`, `monospace`

## 4. Components
**Buttons**
- Primary Button: Background `#3b82f6`, Text white, Radius 8px.
- Secondary Button: Background `#1c2330`, Border `#2a3441`, Text `#e6edf3`.

**Cards & Panels**
- Background: `#161b22`
- Border: `1px solid #2a3441`
- Radius: `10–12px`

**Input Fields**
- Background: `#0d0f14`
- Border: `1px solid #2a3441`
- Focus: `1px solid #3b82f6` (blue glow)
- Text: `#e6edf3`

## 5. Layout Style
- Max width: 1100–1200px
- Centered layout
- Dashboard-style grid
- Split pane for editor tools (Raw Markdown on left, Preview on right)

## 6. Effects
- Very subtle shadows
- Soft glow on focus
- No heavy gradients
- Minimal blur effects
